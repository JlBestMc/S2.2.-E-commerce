## 🛠️ Getting Started

### 1️⃣ Clone this repository

```bash
git clone https://github.com/JlBestMc/S2.2.-E-commerce.git
cd S2.2.-E-Commerce
```

### 2️⃣ Install Dependencies

Make sure you have Node.js installed. Then install the packages:

```bash
npm install
```
